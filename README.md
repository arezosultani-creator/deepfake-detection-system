# AI-Generated Video Detection System (Deepfake Detection)

## Overview
This project detects AI-generated (deepfake) video frames using computer vision
and machine learning techniques.

The system analyzes video frames to identify visual inconsistencies caused by
synthetic manipulation.

## Highlights
- Processed 5,000+ video frames
- Achieved 91% detection accuracy
- Focused on data validation and system reliability

## Technologies Used
- Java
- OpenCV
- DeepLearning4J (DL4J)
- Machine Learning Classification

## Skills & Concepts
- Computer Vision
- Machine Learning
- Data Validation
- Secure & Reliable Systems
